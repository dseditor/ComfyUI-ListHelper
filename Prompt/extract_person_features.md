# 圖片人物特徵提取指令

## 任務說明

你將收到一個寫真雜誌生成提示詞，其中 `{EXTRACT_FROM_IMAGE}` 是一個佔位符。

請執行以下步驟：

### 步驟 1：分析圖片中的人物特徵

仔細觀察圖片，提取以下人物特徵：

1. **國籍/種族特徵**：判斷人物的種族（例如：亞洲人、歐美人、非洲人等）
2. **臉型**：描述臉型（例如：圓臉、瓜子臉、方臉、鵝蛋臉等）
3. **五官特徵**：
   - 眼睛：大小、形狀、是否雙眼皮、眼睛顏色
   - 鼻子：高挺或扁平、鼻型
   - 嘴巴：大小、唇形、唇色
4. **妝容風格**：描述妝容（例如：自然妝、濃妝、裸妝、煙燻妝等）
5. **髮型和髮色**：詳細描述髮型（長短、直捲）和顏色
6. **其他明顯特徵**：眼鏡、飾品、痣、特殊標記等

### 步驟 2：生成簡潔的特徵描述

將提取的特徵整理成簡潔的中文描述（50-80字），格式如下：

```
[種族][性別]，[臉型]，[眼睛特徵]，[鼻子特徵]，[妝容]，[髮型和髮色]，[其他特徵]
```

**範例**：
```
亞洲女性，瓜子臉，大眼睛雙眼皮，高挺鼻梁，自然妝容，黑色長直髮，戴黑框眼鏡
```

### 步驟 3：替換佔位符

在原始提示詞中，找到所有 `{EXTRACT_FROM_IMAGE}` 佔位符，將其替換為你生成的特徵描述。

### 步驟 4：輸出完整提示詞

輸出替換後的完整提示詞，確保：
- 所有 `{EXTRACT_FROM_IMAGE}` 都已被替換
- 特徵描述準確且簡潔
- 保持原始提示詞的其他內容不變

## 重要提示

- **只替換佔位符**：不要修改提示詞的其他部分
- **保持格式**：確保替換後的提示詞格式正確
- **簡潔描述**：特徵描述應該簡潔明瞭，不要過於冗長
- **直接輸出**：直接輸出替換後的完整提示詞，不要包含其他說明文字

## 範例

### 輸入提示詞
```
人物特徵：{EXTRACT_FROM_IMAGE}

請為寫真雜誌生成內容...
```

```
請為寫真雜誌生成內容...
```
